# NAME

reader-categories — list all categories

# SYNOPSIS

**reader categories**

# DESCRIPTION

Prints a table of all categories with their IDs and names.

# OPTIONS

**--help**
: Show help message and exit.

# EXIT STATUS

**0**
: Success.

**1**
: API error or network/server error.

# EXAMPLES

    reader categories
